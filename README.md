# DPP Ghost Theme

Ghost theme for Dreadfully Poor Publishing Co.

## Status

Initial production scaffold managed through ChatGPT/GitHub.

## Local testing

Upload the repository contents as a ZIP through Ghost Admin, or clone locally into a Ghost development install.

## Required Ghost files

- `package.json`
- `default.hbs`
- `index.hbs`
- `post.hbs`
- `page.hbs`
